# 0040 — 工具结果人类可读投影

- 状态：已采纳
- 日期：2026-07-16

## 决策

dgoal 的八个公开工具默认只显示紧凑的动作、状态或数量摘要。用户按 `Ctrl+O` 展开时，显示 `content` 的完整人类可读文本，以及由工具显式提供的 `details.display`（受控展示字段）。`details` 的其他结构化数据仍保留给模型、运行时和测试使用，不得交给宿主默认渲染，更不得以原始 JSON 或运行时对象直接展示。

## 后果

每个公开工具必须配置统一的 `renderResult`，覆盖成功、错误和进行中状态。创建、提案、更新、完成和建检工具应将有价值的展开信息预格式化为 `details.display`；`plan_read` 的完整摘要仍直接来自 `content`。进行中的建检在展开时显示其活性文本。测试须验证折叠摘要、受控展开详情、进行中活性和非白名单字段不泄漏。
