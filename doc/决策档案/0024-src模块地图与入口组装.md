> Status：已实现（vNext，含收敛说明）。

保留 `index.ts` 作为 Pi 注册、事件订阅和模块组装入口；按真实职责惰性迁入 `src/plan`、`src/audit`、`src/isolated-pi`、`src/tui`、`src/runtime`、`src/goal-runtime`、`src/startup`。`isolated-pi` 只承载隔离 Pi 调用、JSON 事件、模型注册表预检和候选选择的公共事实；auditor 与 context summarizer 各自保留不同重试和失败语义。审计用量账本归 `src/audit`（`usage.ts`），实际审核模型与 Goal Repair 展示归 `src/tui`。可变会话状态归 `src/goal-runtime/state.ts` 单例；Pi 注册与事件订阅归 `src/startup/index.ts`。
