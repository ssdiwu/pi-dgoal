# ADR 0023：终审修复作为展示阶段

> Status：已被 ADR 0038 覆盖。Goal check rejected 不再建立专用 Goal Repair 状态或展示投影。

goal 进入 `rejected` 后，持续显示浮层、状态栏与 `/dgoal s` 将展示“终审修复（Goal Repair）· 第 N/3 次”；`paused(audit_failed_3x)` 展示“终审修复已暂停”。终审修复是帮助用户理解“所有 phase 已完成、但 goal 正依据终审反馈修复”的展示投影，不是 phase、task 或新的状态机状态，不进入 plan，也不引入额外建检。
